# 스네이크 게임 (HTML 단일 파일)

키보드 **화살표 키(← ↑ → ↓)**로 플레이하는 클래식 스네이크 게임임.  
한 파일(`snake.html`)로 실행되며, **시작/일시정지/리셋 버튼**, **점수/최고점 기록**, **속도 증가** 기능을 포함함.

## ✨ 특징
- **단일 HTML 파일**: `<style>`과 `<script>`를 한 파일에 포함함  
- **조작**: 화살표 키로 이동, **스페이스바**로 시작/일시정지 토글 가능함  
- **점수/최고점**: 로컬스토리지에 최고점 자동 저장됨  
- **속도 증가**: 점수 4점마다 속도 소폭 증가함  
- **고해상도 선명도**: `devicePixelRatio` 반영하여 캔버스 스케일 보정함  
- **반응형 표시**: 창 크기 변경 시 스케일 유지함
